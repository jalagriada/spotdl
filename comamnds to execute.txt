ON LINUX and macOS:
python -m venv spotify_venv
source spotify_venv/bin/activate
pip install requests
pip install spotdl
pip install -r requirements.txt

ON WINDOWS:
python -m venv spotify_venv
spotify_venv\Scripts\activate
pip install requests
pip install spotdl
pip install -r requirements.txt

Or just do this instead on Linux and macOS:
chmod +x setup.sh   # Give execute permission
./setup.sh          # Run the script
source spotify_venv/bin/activate

For Windows:
Simply run the batch file to automatically create the virtual environment and install all dependencies. After running the script, type "call spotify_venv\Scripts\activate"
